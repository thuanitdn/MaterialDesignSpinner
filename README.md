Simple spinner css3 with material design
*  Using *
          <div class="spinner" data-options="large"  role="progressbar" aria-valuetext="Loading…"></div>
* color *
<style type="text/css">.spinner{color:yourcolor;}</style>  
* size *
data-options="small"
 or
data-options="large"
